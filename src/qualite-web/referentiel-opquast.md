# Le référentiel Opquast

Opquast (pour “Open Quality Standards”) est une entreprise française qui œuvre pour démocratiser la notion de qualité Web. Elle est une référence dans le domaine de la qualité Web.
Elle met en place des checklists sous licence ouverte.
Propose des formations et des certifications autour de ses référentiels sur la qualité Web.

## Un socle minimum et transversal

Il existe de nombreux référentiels dans le domaine du Web :
Le WCAG pour l’accessibilité
Le W3C pour la validité des pages Web
L’optimisation (SEO) pour le référencement
Les tests d’intrusion pour la sécurité
Les test utilisateurs pour l’ergonomie
…
Le référentiel Opquast traite de tous ces sujets à un niveau de base pour doter les professionnels du Web d’un socle de connaissances et d’un vocabulaire communs.

Source : Opquast day 2017 - compte-rendu et bilan

## Modèle VPTCS

Créé par Elie Sloïm et Eric Gateau, VPTCS est un modèle transversal pour la qualité de l’expérience utilisateur.
Il est une représentation simplifiée et partielle de la réalité. Il peut être utilisé pour comprendre, organiser, mesurer et prédire.
En savoir plus : https://www.opquast.com/rendre-le-web-meilleur/vptcs/

## Certification Opquast

Opquast propose une certification pour les professionnels du Web. Elle s’appuie les règles de qualité web Opquast, qui traitent des principaux domaines de risque au sein des projets web et numériques : vie privée, écoconception, mauvaise UX, accessibilité, sécurité, performance, etc.

# Comment établir des critères de qualité ?

::: tip Pour vous, qu'est-ce qu'un site web de qualité ?
Prenez le temps d'y réfléchir.
Si besoin, faites une liste de critères qui vous semblent importants.
:::

## Objectiver les critères

Exigences génériques

Le site est ergonomique, le design est attrayant…

Exigences objectives

Il est possible de revenir à la page d'accueil depuis toutes les pages.
La navigation sur le site ne provoque pas l'ouverture de popups.
Le site propose un moteur de recherche interne.
Etc.

## Rationaliser les critères 

Les critères de qualité doivent être :

Utiles
⛔️ Le site est intégralement traduit dans au moins 3 langues.
✅ Chaque changement de langue est signalé.
Réalistes
⛔️ Tous les liens sont valides.
✅ Tous les liens internes du site sont valides.
Vérifiables
⛔️ Le site est accessible.
✅ Le site est intégralement utilisable au clavier.
Consensuels
⛔️ Les liens pointant vers des sites tiers s’ouvrent dans un nouvel onglet.
✅ Les liens internes et externes sont différenciés.

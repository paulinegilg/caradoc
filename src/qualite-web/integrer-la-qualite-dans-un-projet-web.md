# Intégrer des critères de qualité dans un projet web

les différentes phases d'un projet :

Conception
Rédaction du cahier des charges
Production
Prototypes
Design
Développement
Intégration
Contenus

Exploitation
Suivi du projet
Tierce maintenance applicative (TMA)
Évolutions, corrections

## En phase de conception

Lister les bonnes pratiques à respecter, en sachant que :
Toutes les bonnes pratiques ne peuvent pas être mises en place. Cela peut dépendre du budget et des contraintes du projet.
Toutes les bonnes pratiques ne doivent pas être mises en place. Par exemple, les bonnes pratiques des sites e-commerce pour un site vitrine.

## En phase de production

Mettre en place les bonnes pratiques dans chaque phase de production (prototypes, design, développement, intégration, contenus)

Tester les bonnes pratiques dans chaque phase de production

## En phase d'exploitation

Réaliser des audits de qualité
Ajouter de nouvelles bonnes pratiques dans les évolutions
Tester les nouvelles bonnes pratiques après leur ajout
Améliorer le projet en ajoutant régulièrement de nouvelles bonnes pratiques
